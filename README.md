# Algorithm-Visualizer
A visualizer GUI, for all the sorting and searching algorithms, using tkinter library in python

Sorting Algorithms Supported-
1. Bubble Sort
2. Insertion Sort
3. Selection Sort
4. Merge Sort
5. Counting Sort
6. Quick Sort
7. Heap Sort
8. Genome Sort
9. Radix Sort

Searching Algorithms Supported-
1. A* Pathfinder
